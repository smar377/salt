# Salt

[Salt](https://en.wikipedia.org/wiki/Salt_(software)) (SaltStack) is a Python-based, open-source framework for event-driven IT automation, remote task execution, and configuration management — supporting infrastructure-as-code for system and network deployment, config automation, SecOps orchestration, and hybrid cloud control.

This repo documents my hands-on experimentation with Salt, focused primarily on **network automation**.
